## hotkeys
### Build camera 
* Build-Camera toggle: L

### Hold attack
* Hold-attack toggle: H

### Valheim legends
* First-skill: mouse 4
* Second-skill: left_alt + mouse 4
* Third-skill: G

### BetterUI
* Hud-editor: F7
